# iOS 18 Development
A hands-on SwiftUI course built entirely inside an Xcode project. 37 self-contained lessons – open any file, read the code, run the preview.

<br>

### Requirements
| Tool | Purpose |
|------|---------|
| **Xcode 16+** | Run previews and the full app |
| **iOS 18 SDK** | Required for all lessons |

<br>

### How It Works
Each lesson is a single Swift file with its own `#Preview`. You can:

1. **Follow in order** — lessons are numbered 01 to 37, building on each other
2. **Jump around** — every file runs independently, no wiring needed
3. **Use the app** — run the project and pick any lesson from the toolbar menu

### Tips
- **First time?** Start at [`01-TextAndModifiers.swift`](iOS18-Development/Lessons/01-TextAndModifiers.swift) and work your way up
- **Quick reference?** Jump to any file by number — e.g., [`18-Menus.swift`](iOS18-Development/Lessons/18-Menus.swift)
- **Extending the course?** Add `38-YourTopic.swift` and follow the same pattern
